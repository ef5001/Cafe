# Chaotic Neutral Cafe — Project Context for Claude Code

## What We're Building

A website for **Chaotic Neutral Cafe**, a board game cafe opening in Altavista, VA. This is a brand-new small business and the website should reflect its identity: a welcoming community third space where craft coffee meets board game culture.

### Concept

A craft coffee shop combined with a curated board game library. The goal is to be Altavista's community third space — a town that currently doesn't have one.

**Non-members:** 2 hours of free game play with any food/drink purchase  
**Members ($10/month):**
- Unlimited game time
- One free drink per visit
- Priority reservations
- Members-only events and tournaments
- 10% off food, merch, and events

- Community-forward, not founder-forward (brand-first, not Ethan-first)

## Website Goals

The website should serve multiple purposes:
1. **Pre-opening:** Drive traffic to the Kickstarter, communicate the concept, build excitement
2. **Post-opening:** Serve as the primary public-facing presence (hours, menu, membership info, events)

Build with the pre-opening phase as the immediate priority, but structure it so it can evolve.

### Key Pages / Sections (suggested, open to iteration)
- **Home/Hero** — concept overview, Kickstarter CTA
- **About** — what Chaotic Neutral Cafe is and why Altavista needs it
- **Membership** — explain the model clearly (non-member vs. member)
- **Support Us / Kickstarter** — reward tiers, campaign link
- **Coffee** — brief overview of the coffee program vision (supplier TBD)
- **Games** — brief overview of the game library vision
- **Contact / Location** — 428 Main St, Altavista, VA
